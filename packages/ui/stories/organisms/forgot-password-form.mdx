# ForgotPasswordForm Component

## Overview

The form component includes fields for email and a button to submit email for a password reset email.


## Props

- `email`: The email address entered by the user.
- `errors`: Array of error messages related to the email field.
- `onSubmit`: Function to handle form submission, which sends the email to the server for password reset.

## State

- `emailError`: Sets component error state based on `errors` prop.
