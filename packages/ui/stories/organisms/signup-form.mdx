# SignupForm Component

## Overview

The form component includes fields for email, password, and confirm password, with appropriate client side validation.


## Props

- `email`: The email address entered by the user.
- `password`: The password entered by the user.
- `confirmPassword`: The password confirmation entered by the user.
- `errors`: Array of error messages related to the email, password, and confirm password fields.
- `onSubmit`: Function to handle form submission, which sends the email and password to the server for account creation.
- `onEmailChange`: Function to handle changes in the email input field.
- `onPasswordChange`: Function to handle changes in the password input field.
- `onConfirmPasswordChange`: Function to handle changes in the confirm password input field.

## State
- `emailError`: Sets component error state based on `errors` prop.
- `passwordError`: Sets component error state based on `errors` prop.
- `confirmPasswordError`: Sets component error state based on `errors` prop.
