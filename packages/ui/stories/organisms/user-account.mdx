# AvatarImage Component

## Overview

- User account component that displays user information and allows logout. 
- Non Anonymous users can send themselves a password reset email.
- Anonymous users see their anonymous email and password.



## Props
- `useremail`: The email address of the user.

## State

- `isAnon`: Boolean indicating if the user is anonymous.
