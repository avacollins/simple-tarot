# QuickNav Component
## Overview
The QuickNav component provides a floating action button that opens a speed dial menu with options for starting a new reading and navigating to profile, history, new reading, and home screens.

## Props

This component has no props.

## State
- `open`: Boolean state to track if the speed dial is open or closed.
