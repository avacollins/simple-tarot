# ShuffleCard Component
## Overview
The ShuffleCard component displays a single Card component in a shuffled state.
It uses `react-native-reanimated` for animations.

## Props
- `cardIndex`: The index of the card to display.
- `isShuffling`: Boolean indicating if the card is currently being shuffled.
- `cutCards`: Function to handle cutting cards.
- `cutCardIndex`: The index of the card to cut.

## State
None.
